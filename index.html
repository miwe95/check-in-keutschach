Check in Instructions / Check in Anweisungen


1. Parken/Parking

Parken ist auf der Wiese vor dem Haus möglich.
Parking in front of the house.

2. Weg zur Tür/Path to the door

Einfach dem Weg zur weißen Tür folgen.
Just follow the path to the white door.

3. Aktivieren des Schlosses/Activating the Lock

Auf der Tür findet man ein elektronisches Nummernschloss. Um das Schloss zu aktivieren bitte auf das kleine "offne Schlossicon" drücken.
To activate the lock please press the "open lock icon".

4. Code eingeben/insert code

Auf dem kleinen Bildschirm des Schlosses erscheint das Datum. Das Schloss ist aktiv und der Code, der per Airbnbchat mitgeteilt wurde kann nun eingegeben werden.
The date appears on the little screen. The lock is now active and the code which was send via Airbnbchat can be inserted.

5. Code bestätigen/Confirm code

Erneut auf das kleine "Schlossicon" drücken um den Code zu bestätigen.
To confirm the Code press the lock icon again.

6. Drehen des Schlosses/Turn the lock

Wurde der Code richtig eingeben erscheint auf dem Bildschirm der Text "Turn handle". Nun kann das Schloss wie ein Schlüssel nach rechts gedreht werden um die Türe zu entsperren.
If the code was inserted correct you can now turn the lock like a key to the right to unlock the door.

7. Schließen der Tür von außen/Lock the door from outside

Möchten Sie das Haus verlassen und die Türe von außen schließen, einfach die vorherigen Schritte wiederholen und das Schloss nach rechts drehen.
If you want to lock the door from the outside just repeat the steps before and turn the lock to the left.

8. Videoanleitung/Video instructions

Video zeigt, wie das Schloss geöffnet wird. Bitte benutzen Sie den Code der Ihnen gesendet wurde.
Video showing how the door can be opened. Please use the sent code.
